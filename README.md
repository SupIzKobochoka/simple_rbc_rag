Небольшой пет проектик по RAG системе.
Бот, отвечает на вопросы пользователей, используя базу новостей rbc. Отдельно передаются новейшие новости и старые

Данные - новости rbc за 2000-2020 года (при желании можно спрасить более новые)  
Модель - mimo-v2-flash (как раз пока она по api доступна бесплатно)  
БД - opensearch  
Фронт - телеграм бот (полностью написано ботом, но работает)  
Эмбединги делаются локально
